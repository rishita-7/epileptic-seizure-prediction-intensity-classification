# Real-Time Epileptic Seizure Prediction

##  Overview

This project aims to build a research-grade machine learning pipeline for real-time epileptic seizure prediction and intensity classification using EEG signals from the CHB-MIT dataset.

##  Key Features

* Memory-efficient EEG preprocessing (MNE)
* Seizure prediction (pre-ictal vs inter-ictal)
* Seizure intensity classification
* Modular and reproducible pipeline

##  Project Structure

* `src/` → Core pipeline code
* `data/` → Raw, interim, processed (ignored in Git)
* `scripts/` → Execution scripts
* `notebooks/` → Experiments

##  Current Phase

Phase 1: Data loading and preprocessing pipeline

##  Goal

Develop a real-time deployable system for early seizure prediction with severity estimation.
